This code is written using MATLAB 2022b, Yalmip and Cplex 12.10.0. 
The 'inxy3' is the longitude and latitude data of grid cells. 
The two 'Main' files are the optimization algorithms. 
The essential wind speed dataset covering the entire South China Sea region is available in the file data_v_NH.mat. It can be downloaded via: https://pan.baidu.com/s/1kMVhi80zjaEJoeIXOAcjDA (Access code: uk48).
